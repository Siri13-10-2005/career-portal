# Career Portal

A full-stack Career Portal built using React, FastAPI, MongoDB, and JWT Authentication. The platform enables students to discover jobs, apply for positions, track applications, and receive notifications, while recruiters can manage job postings, applicants, and hiring analytics.

## Features

### Student Features

* User Registration & Login
* JWT Authentication
* Browse Available Jobs
* Search Jobs
* View Job Details
* Apply for Jobs
* Save Jobs
* View Saved Jobs
* Track Applications
* Notifications
* Student Analytics Dashboard

### Recruiter Features

* Create Jobs
* Edit Jobs
* Delete Jobs
* View Applicants
* Shortlist Applicants
* Reject Applicants
* Recruitment Analytics Dashboard
* Applicant Management

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
